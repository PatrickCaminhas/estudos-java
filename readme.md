# Projeto de aprendizado em Java

Este é um projeto em java para relembrar caracteristicas importantes em java e na programação. Escolhi java pois é uma linguagem de tipagem forte e que me restringe em muitas situações me ajudando a evitar "gambiarras".

## Conteúdo

O conteúdo atual desse projeto é uma calculadora financeira atualmente, passando por calculo de salario liquido, impostos retidos na fonte, gastos recorrentes, gastos eventuais, outras formas de renda e relatorio completo.

## Etapas
As etapas são definidas por cada forma na qual será feito esse projeto, sendo elas em terminal, desktop e web.

1. Terminal **[Etapa Atual]**: A interação é feita com usuário via o terminal do sistema operacional, utilizando laços de repetição, funções de limpeza de terminal e condicionais para formar a tela de interação do usuário.

2. Desktop: A interação será feita com usuário via tela construida em JavaFX.

3. Web: O servidor será feito via sistema web construido com framework Spring Boot e a interação será feita com um framework front-end e framework CSS.

#### Quais frameworks front-end possiveis de utilizar
1. React
2. Vue.js

#### Quais frameworks CSS possiveis de utilizar:
1. Tailwind CSS
2. Bootstrap
